## 목적
> 과학사를 **알고 싶은 학생**들이나, **과학에 흥미가 많은 학생**들을 위해, **과학의 시작부터 현재까지의 역사를 보여주기 위함**

## 어떻게 제작할 것인가?
필수적으로 해야 할 것들:
- 위키백과나 다른 저명성 있는 홈페이지로부터 과학사 정보 수집
- 한국어로 번역 및 일부 표현들을 가공
